# Job Portal 🚀

A full-stack Job Portal web application built using the MERN Stack that connects job seekers with recruiters through a modern and efficient hiring platform.

## 📌 Overview

This platform is designed to simplify the recruitment process for both candidates and employers. Job seekers can create profiles, upload resumes, search for jobs, and apply easily. Recruiters can post job openings, manage listings, and review applications in one place.

## ✨ Features

### 👨‍💼 For Job Seekers
- User registration and login
- Create and update profile
- Upload resume
- Browse available jobs
- Search and filter jobs
- Apply for jobs
- Track applications

### 🏢 For Recruiters
- Recruiter registration and login
- Post new jobs
- Update or delete job posts
- View applicants
- Manage job listings

### 🔐 Core Functionalities
- Role-based authentication
- Secure login system
- Responsive UI design
- REST API integration
- Smooth user experience
